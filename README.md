# 🖼️ INF01046 - Image Processing

Project developed for the **INF01046 - Fundamentals of Digital Image Processing** course at UFRGS. This implementation uses **GTK+ 3** for the graphical user interface and **OpenCV** for video processing.

---

## 🛠️ Prerequisites (Arch based systems)

To compile and run this project, you need the GTK3 development libraries and OpenCV.

Install the required dependencies using `pacman`:

```bash
sudo pacman -S gtk3 opencv pkgconf cmake
```

To run `photoshop`

```bash
cd photoshop
make run
```

To run `video-editting`
```bash
cd video-editting
make run
```
